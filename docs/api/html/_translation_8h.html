◆ ctTranslationSetLanguage()

ctResults ctTranslationSetLanguage ( const char * isoCode)

Set the target language to be loaded next game update.

Parameters
isoCodeFormat as follows: "(ISO 639-1)-(ISO 3166 A2)" (ex: "en-us")

The translation system will attempt to load the region specific mo file in the target language first, if this fails it will just look for the language. Full RFC 4646 is not supported.
